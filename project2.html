Project

    Top 10 Airline Review Analysis
    Using Python

    Exploratory Data Analysis (EDA), K-Means Clustering, NLP, Decision Tree, Random Forest

    Obejective

    1. Key Drivers Identification: To determine key factors that most significantly influence customer ratings within the airline industry, and how these factors influence it.

    2. Sentiment Analysis: To perform sentiment analysis on the review dataset in order to classify customer opinions and identify prevailing emotional trends.

    3. Prediction: To develop predictive models to forecast customer satisfaction based on identified influencing factors and sentiment trends extracted from review data.

    Dataset Overview

    This dataset was originally sourced from Kaggle. It contains reviews of the top 10 rated airlines in 2023 sourced from the Airline Quality. The reviews cover multiple elements of the flight experience, such as seat comfort, cabin crew service, food and drinks, inflight entertainment, value for money, and overall satisfaction.

    Insights from Query Analysis

    1. Company Status: 71% active, 12% acquired, 17% inactive, <1% public, highlighting startup survival trends and exit outcomes.

    2. Consumer, Fintech, and Healthcare industries each have over 510 active companies, indicating strong presence and attractiveness to YC investors.

    3. Product-focused design and AI integration are emerging as core strategic priorities across leading startup sub-industries.

    4. The United States remains the dominant hub for startups, while emerging ecosystems in India, the United Kingdom, and Canada are gaining momentum and present new opportunities for global expansion.

    5. Educational institutions like Stanford, MIT, and UC Berkeley play a critical role in nurturing entrepreneurial talent, suggesting a strategic advantage in building founder networks from these ecosystems.

    6. Many startup founders have experience at tech giants like Google and Facebook, highlighting the strong influence of career backgrounds at leading firms.

		        Status, COUNT(*) AS Count,

		    FROM YCombinator.companies

		    ORDER BY Count DESC;

    2. Leading Industries Among Active Companies

    		    SELECT Industry, COUNT(companies.Company_ID) AS NoofCompanies
		    FROM my_schema3.companies
		    ON companies.Company_ID = industries.Company_ID
		    WHERE Status <> "Inactive"
		    GROUP BY Industry
		    ORDER BY NoofCompanies DESC
		    LIMIT 5;

    3. Leading Sub-Industriess

    		    SELECT Sub_Industry, COUNT(companies.Company_ID) AS NoofCompanies
		    FROM my_schema3.companies
		    LEFT JOIN my_schema3.industries
		    ON companies.Company_ID = industries.Company_ID
		    WHERE Status <> "Inactive"
		    GROUP BY Sub_Industry
		    HAVING Sub_Industry <> ""
		    ORDER BY NoofCompanies DESC
		    LIMIT 5;

    4. Regional Distribution of Startups

    		    SELECT 
			CASE 
			    WHEN region.Region IS NULL OR region.Region = '' THEN 'Unknown'
			    ELSE region.Region
			END AS Region,
			CASE 
			    WHEN region.Country IS NULL OR region.Country = '' THEN 'Unknown'
			    ELSE region.Country
			END AS Country,
			COUNT(companies.Company_ID) AS Num_Companies
	             FROM final_project_schema.region
		     LEFT JOIN final_project_schema.companies
		     ON region.Company_ID = companies.Company_ID
		     WHERE companies.Status <> "Inactive"
		     GROUP BY Region, Country
		     ORDER BY Num_Companies DESC
		     LIMIT 10;

    5. Top Schools for Startup Founders

    		     SELECT school, COUNT(school) AS cschool
		     FROM (
		     SELECT school.HN_ID, school.School
		     FROM school
		     JOIN (
		     SELECT companies.Company_ID, founders.HN_ID, companies.Slug, companies.Status, founders.First_Name, 
	             founders.Last_Name
		     FROM companies
		     RIGHT JOIN founders
		     ON companies.Slug = founders.Company_Slug
		     WHERE companies.Status <> 'Inactive'
		     ORDER BY companies.Company_ID
		     ) AS founder_noninactive_t
		     ON school.HN_ID = founder_noninactive_t.HN_ID
		     ORDER BY HN_ID
		     ) AS noninactive_school_t
		     GROUP BY school
		     ORDER BY cschool DESC
		     LIMIT 5;

    6. Work Experience of Startup Founders

    		     SELECT company_experience.Company, COUNT(founders.HN_ID) AS Num_Founders
		     FROM company_experience
		     JOIN founders 
		     ON company_experience.HN_ID = founders.HN_ID
		     JOIN (
			  SELECT companies.Company_ID, companies.Slug
			  FROM companies
			  WHERE companies.Status <> 'Inactive'
		     ) AS active_companies
		     ON founders.Company_Slug = active_companies.Slug
		     GROUP BY company_experience.Company
		     ORDER BY Num_Founders DESC
		     LIMIT 10;
